## 📌 项目简介

本项目是一个校园体育场馆预约系统，覆盖学生端与管理员端的完整业务流程，  
模拟真实校园体育管理平台的运行逻辑。

系统支持：

- 场地预约与组队预约  
- 钱包与信用体系  
- 运动社区互动  
- 管理员后台管理  

适用于课程设计、教学实践、项目展示与答辩。

---

## 🚀 系统功能

### 👨‍🎓 学生端功能

- 场地检索：按校区 / 场馆 / 项目筛选  
- 时间表展示：实时显示各时段状态  
- 预约规则：  
  - 只能预约未来 7 天  
  - 不能预约过去时间  
- 预约模式：  
  - 个人预约  
  - 组队预约（队员自动同步）  
- 我的预约：  
  - 查看历史  
  - 取消并填写原因  
- 校园钱包：  
  - 充值、扣费  
  - 查看流水  
- 信用体系：  
  - 信用分影响预约权限  
- 社区系统：  
  - 发帖、评论、删除自己评论  
  - 点赞 / 取消点赞  
  - 评论区卡片化美化  
- 报修系统  
- 失物招领  
- 公告与封场通知  

---

### 🛠 管理员端功能

- 场馆与场地管理  
- 封场事件管理（维修 / 赛事 / 课程）  
- 器材管理：  
  - 修改价格  
  - 修改库存  
  - 修改图片  
- 预约全览与违规处理  
- 信用扣分与处罚  
- 财务统计与流水审计  
- 报修处理  
- 社区内容审核
